passport를 이용한 인증방식에서 backend와 frontend의 도메인 주소가 다르기 때문에 쿠키가 연결되지 않는다. 이를 해결하기 위해서

axios.get이나 axios.post를 사용할때 마지막 인자값에 { withCredentials: true } 를 추가해주면 인식한다.

ex) axios.get('some api url', {withCredentials: true});

	axios.post('/api/login/signUp', { 
   	     user: this.user
   	   }, { withCredentials: true })

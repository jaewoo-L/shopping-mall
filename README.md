# shopping-mall
Address:https://jaewoomall.herokuapp.com/  

## Project Stack  
### Client
Following items are core frontend technologies used in this project:
- Vue.js
- Vue-cli
- Vue-router
- Vuex
- Axios

### Server
Following items are core backend technologies used in this project:
- Node.js
- Express
- MongooseDB
